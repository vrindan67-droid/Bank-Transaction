Bank Transaction Fraud Detection System

A machine learning-based project developed to identify fraudulent bank transactions using transaction data analysis and predictive modeling. This project uses Python, Pandas, Matplotlib, and Scikit-learn to analyze transaction patterns and classify transactions as Fraud or Not Fraud.

📌 Project Overview

This project focuses on detecting suspicious bank transactions by analyzing transaction amount and fraud indicators from a banking dataset. The system performs:

Data preprocessing
Fraud analysis
Data visualization
Machine learning model training
Fraud prediction for new transactions
🚀 Technologies Used
Python
Pandas
Matplotlib
Scikit-learn
Excel Dataset
📂 Features
Import and analyze bank transaction dataset
Visualize fraud vs non-fraud transactions
Train Logistic Regression model
Predict fraudulent transactions
Interactive dashboards using Excel and Power BI

Fraud vs Non-Fraud comparison

📈 Machine Learning Model

The project uses Logistic Regression for fraud detection:

Dataset split into training and testing sets
Model trained using transaction amount
Accuracy calculated using Scikit-learn metrics
